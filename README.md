# Polarize

MATLAB framework for polarization calculations in periodic molecular crystals.

Current status:
- crystal/supercell builder
- unique molecule bookkeeping
- active/remove molecule selection
- charge-pattern assignment
- nonperiodic external field and Thole SCF
- direct and iterative solver comparison

Next step:
- periodic dipole interaction matrix via Ewald assembly

Main test scripts:
- run/run_single_config.m
- run/run_compare_solvers.m
